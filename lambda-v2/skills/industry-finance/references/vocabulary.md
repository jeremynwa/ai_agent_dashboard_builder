# Vocabulaire Finance (FR)

## Termes a utiliser

| Terme correct (FR) | Ne PAS utiliser | Contexte |
|---|---|---|
| Chiffre d'affaires (CA) | Revenue, Sales | Total des ventes |
| Charges d'exploitation | Operating expenses | Couts operationnels |
| Resultat d'exploitation | Operating income | CA - Charges d'exploitation |
| Resultat net | Net income, Profit | Benefice apres impots |
| Marge brute | Gross margin | (CA - Cout des ventes) / CA |
| Marge nette | Net margin | Resultat net / CA |
| Tresorerie | Cash, Treasury | Liquidites disponibles |
| Capitaux propres | Equity | Fonds propres de l'entreprise |
| Dettes | Liabilities, Debt | Engagements financiers |
| Actif | Assets | Biens et droits |
| Passif | Liabilities (bilan) | Dettes + Capitaux propres |
| Exercice fiscal | Fiscal year | Annee comptable |
| Trimestre | Quarter | Periode de 3 mois (T1, T2, T3, T4) |
| Dotations aux amortissements | Depreciation | Perte de valeur comptable |
| Provisions | Provisions | Reserves pour risques |
| Creances clients | Accounts receivable | Montants dus par les clients |
| Dettes fournisseurs | Accounts payable | Montants dus aux fournisseurs |
| Besoin en Fonds de Roulement | Working capital | BFR = Stocks + Creances - Dettes CT |

## Labels de pages

- Vue d'ensemble (pas "Overview" ou "Dashboard")
- Analyses financieres (pas "Financial Analysis")
- Parametres (pas "Settings")

## Labels de filtres

- Exercice (pas "Year")
- Periode (pas "Period")
- Entite / Filiale (pas "Entity")
- Poste de charges (pas "Expense category")

## Formatage des nombres

- Montants : `1 234 567 EUR` ou `1,2M EUR` (pas de $ sauf si donnees en dollars)
- Pourcentages : `42,5%` (virgule decimale, pas point)
- Ratios : `0,85` (virgule decimale)
