# Vocabulaire SaaS / Tech (FR)

## Termes a utiliser

Les termes SaaS sont souvent utilises en anglais meme en contexte francophone. Utiliser le terme anglais avec l'explication FR entre parentheses au premier usage.

| Terme | Explication FR | Contexte |
|---|---|---|
| MRR | Revenu mensuel recurrent | Metrique principale |
| ARR | Revenu annuel recurrent | MRR * 12 |
| Churn rate | Taux d'attrition | Clients perdus / Total |
| LTV | Valeur vie client | Revenu total par client |
| CAC | Cout d'acquisition client | Depenses / Nouveaux clients |
| NPS | Score de recommandation | Promoteurs - Detracteurs |
| ARPU | Revenu moyen par utilisateur | MRR / Clients |
| DAU / MAU | Utilisateurs actifs quotidiens / mensuels | Engagement |
| Trial | Essai gratuit | Periode d'evaluation |
| Conversion | Conversion essai â†’ payant | Passage a l'abonnement |
| Onboarding | Integration | Premiere prise en main |
| Upsell | Montee en gamme | Passage a un plan superieur |
| Downgrade | Retrogradation | Passage a un plan inferieur |
| Retention | Retention | Clients conserves |
| Cohorte | Cohorte | Groupe de clients par date d'inscription |
| Expansion | Expansion | Revenu additionnel des clients existants |
| Contraction | Contraction | Baisse de revenu des clients existants |

## Labels de pages

- Vue d'ensemble (pas "Overview")
- Analyses (pas "Analytics")

## Labels de filtres

- Periode (pas "Period")
- Plan / Offre (pas "Tier")
- Cohorte (pas "Cohort")

## Formatage

- MRR/ARR : `125K EUR` ou `1,5M EUR`
- Churn : `2,3%` (virgule decimale)
- NPS : `+42` (avec signe)
- Ratio LTV/CAC : `3,2x`
