# App Factory — Frontend

AI-powered dashboard generator. Upload CSV/Excel or connect a database, write a prompt, and an AI agent generates a complete React dashboard with live preview.

## Quick Start

```bash
npm install
npm run dev
```

Requires `frontend/.env` with API URLs and Cognito config (auto-generated by `lambda-v2/deploy.ps1`).

## Stack

- React 18 + Vite 7
- WebContainer (StackBlitz) for live preview
- Recharts for charts
- Motion.dev for animations
- AWS Cognito for auth
- Custom design system (ds.css, no Tailwind)

## Key Files

- `src/App.jsx` — Main app: prompt input, generation pipeline, preview
- `src/services/api.js` — API client with JWT
- `src/services/files-template.js` — WebContainer base files (includes ds.css)
- `src/services/auth.js` — Cognito auth service
- `src/services/export.js` — Zip export
- `src/components/` — AuthProvider, Login, MatrixRain, FileUpload, DbConnect

## Backend

See `lambda-v2/` for the Lambda backend (SAM). The generate Lambda supports two prompt modes:

- **Standard**: Monolithic system prompt in `generate/index.mjs`
- **Agent Skills (beta)**: Modular `dashboard-generator` skill uploaded to Anthropic (controlled by `USE_BETA_API` + `DASHBOARD_SKILL_ID` env vars)

Deploy: `cd lambda-v2 && .\deploy.ps1`
